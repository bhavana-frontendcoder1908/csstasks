What is CSS??

CSS is a Stylesheet Language

CSS:Cascading Style Sheets

The present version we are using is CSS3

We use css to describe the visual presentation and to control layouts colors and etc etc

There are 3 ways to insert css into html pages

There are:

  1. Inline Styling
  2. Internal Styling
  3. External Styling
1.Inline Styling

In inline styling we use css tag's (or) style's tags are written within the head's tag itself

2.Internal Styling

In internal styling we use css tag's (or) style's tags are written b/w in the head's tag

3.External Styling

In external styling we use css tag's (or) style's tags are written separately with .html and .css extentions and link those files with link tags

We use different types of selectors for applying css in different html pages for different styles; And There are different types of Selectors for applying css like Simple Selectors are A simple selector in CSS refers to the most basic form of selecting elements on a webpage. It can target elements based on various criteria like tag name, class, id, attributes, or their relationship with other elements

Simple Selectors

  1. Element Selectors
  2. Universal Selectors
  3. Id selectors
  4. Class selectors
  5. Grouping selectors

element selectors are also called tag selectors or type selectors and their selects HTML elements based on the element name like

3.Class Selectors
